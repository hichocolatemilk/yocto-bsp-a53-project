# SDK 및 Cross Compile을 통한 Application 배포

## 1. 목표
- Yocto로 빌드한 이미지 환경에서 SDK를 생성하고,  
- 크로스 컴파일을 통해 애플리케이션을 빌드 및 타겟 보드에 적용하는 과정을 수행.

---
## 2. 수행 내용
### 1) SDK 생성
```bash
# poky/build 에서 수행
bitbake core-image-minimal -c populate_sdk
```
#### SDK 환경 설정
```Bash
source /opt/custom-a53/1.0/environment-setup-cortexa53-poky-linux
```
- 크로스 컴파일러, 라이브러리, include 경로 자동 설정
- `$CC` 등 컴파일러 환경 변수 활성화
